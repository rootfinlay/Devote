# Devote
Devote - Bringing democracy back to the people

# First things first
```console
# Clone the repo
$ git clone https://github.com/rootfinlay/Devote.git

# Cd into directory
$ cd Devote

# Install requirements
# pip3 install -r requirements.txt
```

# Run the server
```console
$ python blockchain.py
```

# Run the clients
```console
# To submit a vote
$ python client.py

# To check a vote
$ python client2.py
```
